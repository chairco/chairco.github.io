About Me

我是 Yi-Chieh Chen，但通常朋友們都會稱呼英文名 Jason。但不管是什麼原因讓彼此見面，都很歡迎直接稱呼我 Jason。

這個部落格建立於 2016 年，內容都是分享軟體與開發相關主題。會建立這個部落格最主要是受到 Python 裡認識社群朋友 Liang2 影響。早期並不是很習慣在部落格上記錄在軟體開發上的過程，也不太會去細探每個方法背後更多的原理以及知識，很多累積的經驗並未讓我突破，一直停留在遇到一個問題然後解決一個問題，但解決後過陣子也就遺忘了。不過讀過 Liang2 的文章後，卻開始有一些想法: 軟體的領域實在很廣，要解決的問題也很多。但道理往往是殊途同歸，真正花時間去探究每一次經驗或錯誤背後的知識與原理才能讓自己不斷超越與突破。這也是我開始撰寫部落格文章的原因。

目前喜愛使用 Python 開發軟體，最常使用 Django 進行網頁後端程式的開發。致力於程式與資料庫效能提升(例如協程、並行開發等或是 SQL 語言優化)。閒暇時間喜歡解解 LeetCode 上的題目與研究資料結構與演算法的書籍。偶爾則會研讀機器學習相關數學的論文。希望可以加入新創團隊或能前往美國、歐洲地區或是日本等國家與不同國家的團隊合作開發軟體。

如果正在尋找夥伴一起工作，或是公司內薦機會，也對我的經歷感到興趣，請來信 chairco@gmail.com 告訴我。
