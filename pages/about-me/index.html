About Me

你好，我是 Jason(chairco) 畢業於國立中正大學資訊管理與醫療資訊管理研究所。目前在企業內擔任軟體開發工程師。

專長與經歷

我的專長是軟體開發與統計分析。過去專案主要為開發電子產品的測試軟體，包含自動化測試、透過 SDK 開發與底層韌體(firmware)或是驅動程式(driver)溝通的測試程式並且將所有測試流程做整合。目前則是擔任全端工程師，負責企業內部 IT 資訊系統開發。包含 FrontEnd 的 Web Develope 與 BackEnd 的資料流與演算法。

碩士論文醫院評鑑知識管理系統成功模式之驗證是透過建置 Model 理論來驗證醫院的知識管理系統，使用了很多統計分析技術如假設檢定、迴歸分析、結構方程模式等。因此統計分析為我的第二專長。

興趣

興趣是當一個背包客，已經走過日本、香港、中國等地區，未來計劃走過美國與歐洲各國體驗不同的文化。
