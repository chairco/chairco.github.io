作者簡介

Yi-Chieh 目前擔任資料工程師一職，對機器學習與深度學習也保持著強烈的熱情。過去在軟體自動化開發、系統流程設計與網站前後端開發(通常是後端多一些)有相當經驗。在別人眼裡他是一開口就讓人想睡覺的阿宅，不過會一直努力讓大家不要對他的話題感到太無聊。對了，他非常喜歡小動物，像是貓、狗，小動物們總是很熱情的與他交流。

使用 Python 撰寫軟體與 Django 框架開發網頁後端程式。工作主要專注於程式與資料庫效能提升(例如使用協程、並行開發等或是將 SQL 語言優化)。閒暇的時間喜歡解 LeetCode 上的題目或研究書上資料結構與演算法

如果此時您正在尋找夥伴一起工作，或是願意分享內薦機會，也對我的經歷表示肯定，不管是國內外工作機會，誠摯希望透過 chairco@gmail.com 來信給我 ;)
